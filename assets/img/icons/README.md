# Iconos PWA para CardioSmart IPS

## Iconos necesarios

Esta carpeta debe contener los iconos de la aplicación PWA en diferentes tamaños.

### Tamaños requeridos:
- icon-16x16.png
- icon-32x32.png
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Cómo generar los iconos

### Opción 1: Herramientas online (Recomendado)
1. Ir a [PWA Builder Image Generator](https://www.pwabuilder.com/imageGenerator)
2. Subir el logo de CardioSmart (preferiblemente 512x512px o mayor)
3. Descargar el paquete completo de iconos
4. Extraer en esta carpeta

### Opción 2: Usando el logo de CardioSmart
1. Abrir el logo de CardioSmart en un editor de imágenes (Photoshop, GIMP, etc.)
2. Crear un canvas cuadrado con fondo blanco o transparente
3. Centrar el logo dejando un pequeño padding (10-15%)
4. Exportar en los tamaños requeridos
5. Nombrar según la lista anterior

### Opción 3: Real Favicon Generator
1. Ir a [Real Favicon Generator](https://realfavicongenerator.net/)
2. Subir el logo de CardioSmart
3. Configurar para iOS, Android, etc.
4. Descargar y extraer aquí

## Características del icono ideal

- **Fondo**: Blanco (#FFFFFF) o Azul CardioSmart (#0066CC)
- **Logo**: Centrado con padding del 10-15%
- **Formato**: PNG con transparencia cuando sea posible
- **Resolución**: Alta calidad sin pérdida

## Nota

Los iconos son esenciales para:
- Instalación como PWA en dispositivos móviles
- Pantalla de inicio (home screen)
- Splash screen al abrir la app
- Barra de tareas y notificaciones

Una vez agregados los iconos, la PWA estará completamente funcional.
